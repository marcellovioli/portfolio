# Portfolio Marcello Violi

Questo è un portfolio statico in HTML, CSS e JavaScript.

## Come usarlo
1. Apri `index.html` nel browser.
2. Modifica testi, progetti e contatti dentro `index.html`.
3. Cambia colori e stile dentro `style.css`.
4. Pubblicalo gratis su Netlify o GitHub Pages.

## Struttura
- `index.html`: contenuto del sito
- `style.css`: grafica
- `script.js`: animazioni
- `assets/`: immagini
